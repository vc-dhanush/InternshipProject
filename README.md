Skill-Exchange – Android App using GenAI
📌 Project Overview    https://vc-dhanush.github.io/tempforme/

Skill-Exchange is an Android application designed for rural and local technicians to exchange services using a barter-based system instead of money. The platform helps skilled workers such as plumbers, electricians, carpenters, and mechanics collaborate by trading their skills and earning trust within the community.

This project was developed as part of the MindMatrix VTU Internship Program.

🚀 Problem Statement

Village technicians and skilled workers often need help from others but may not have enough cash to pay for services. Many people are willing to exchange their skills, but there is no proper platform to connect them.

Example:
A plumber can repair an electrician’s pipes in exchange for electrical repair work.

The app solves this problem by creating a community-driven skill exchange ecosystem.

💡 Vision

The main goal of the project is to:

Encourage self-employment
Promote local skill utilization
Build trust and collaboration among workers
Create a self-reliant community using barter-based services

#�� Features

🔹 Skill Profile

Users can:

Create profiles
Add their skills
Mention services they can provide
🔹 Need Post

Users can post requests like:

“Need help with leaking roof.”

🔹 Swap Offer

Users can offer their own skills in exchange for another service.
Example:

“I can provide woodworking in exchange for roof repair.”

🔹 Trust Score
Users gain trust points after successful exchanges
Helps build a reliable community
🔹 Real-Time Skill Board
Displays available skills and requests
Updates dynamically
🔹 Chat-Based Negotiation
Users can communicate and negotiate swaps
